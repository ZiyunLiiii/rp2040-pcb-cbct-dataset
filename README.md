# RP2040-PCB-CBCT Dataset

Cone-beam CT dataset of an Adafruit RP2040 PCB enclosed in a plastic case, designed for metal artifact reduction and plastic-metal beam-hardening correction evaluation.

This repository contains the splash page for the dataset.

Website:

https://ziyunliiii.github.io/rp2040-pcb-cbct-dataset/
