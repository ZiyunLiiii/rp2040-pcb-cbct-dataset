RP2040-PCB-CBCT Dataset

A newly constructed cone-beam CT dataset for plastic-metal beam-hardening correction, scanned from an Adafruit RP2040 PCB enclosed in a plastic case using a North Star Imaging 1420-DR system.

Overview

Built for plastic-metal beam-hardening correction

The scanned object combines low-density plastic with multiple dense metallic components, producing severe streaking, shading, and material-dependent beam-hardening artifacts that are representative of practical industrial inspection problems.

Preprocessing

Minimal MBIRJAX preprocessing example

The example below keeps the dataset-specific preprocessing path from the experiment script: download/extract an NSI scan, compute the sinogram and geometry parameters, crop the cone-beam sinogram, initialize the model, generate transmission weights, and save FDK/MBIR outputs. 

import os
import numpy as np
import jax.numpy as jnp
import mbirjax as mj
import mbirjax.preprocess as mjp


dataset_url = "https://www.datadepot.rcac.purdue.edu/bouman/elec_board/Horizontal_Elec_Board_1800.tgz"
download_dir = "./demo_data"
output_dir = "./outputs"
os.makedirs(output_dir, exist_ok=True)

dataset_dir = mj.download_and_extract(dataset_url, download_dir)

sino, cone_beam_params, optional_params = mjp.nsi.compute_sino_and_params(
    dataset_dir,
    downsample_factor=[2, 2],
    subsample_view_factor=2,
)

sino_cropped, cone_beam_params, optional_params = mjp.auto_crop_sino_conebeam(
    sino,
    cone_beam_params,
    optional_params,
)
sino_cropped = np.maximum(sino_cropped, 0)

ct_model = mj.ConeBeamModel(**cone_beam_params)
ct_model.set_params(**optional_params)
ct_model.set_params(sharpness=1.0, verbose=1, positivity_flag=True)

weights = ct_model.gen_weights(sino_cropped, weight_type="transmission_root")

fdk = ct_model.direct_recon(sino_cropped)
jnp.save(os.path.join(output_dir, "recon_horiz_1800_fdk.npy"), fdk)

mbir_recon, _ = ct_model.split_sino_recon(sino=sino_cropped, weights=weights)
np.save(os.path.join(output_dir, "mbir_recon_horiz_1800.npy"), mbir_recon)

mar_recon = mjp.recon_plastic_metal(
    ct_model,
    sino=sino_cropped,
    weights=weights,
    num_BH_iterations=3,
    num_metal=1,
    num_constraint_update_iter=0,
    order=3,
    verbose=1,
    alpha=1,
    beta=0.002,
    gamma=0.1,
)
jnp.save(os.path.join(output_dir, "recon_horiz_1800_plastic_metal.npy"), mar_recon)
